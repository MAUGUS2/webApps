📋 Listas em HTML

• Listas Não Ordenadas (<ul>)

Usadas quando a ordem dos itens não importa. Cada item é marcado com um símbolo (por padrão, um ponto).

  • 🍎 Maçã
  • 🍌 Banana
  • 🍇 Uva

🔹 Tipos de marcadores:

Você pode personalizar os marcadores usando o atributo type:

  • Marcador circular
  • Marcador quadrado

• Listas Ordenadas (<ol>)

Usadas quando a ordem dos itens é importante (ex: passo a passo, ranking).

  1. 🔧 Abrir o terminal
  2. 📂 Navegar até a pasta
  3. ▶️ Rodar o programa

🔸 Tipos de numeração:

O atributo type permite escolher diferentes estilos de contagem:

  1. Letra maiúscula
  2. Segunda letra
  1. Letra minúscula
  2. Outra letra
  1. Numeração romana maiúscula
  2. Outra etapa
  1. Numeração romana minúscula
  2. Outro item

• Listas Aninhadas (listas dentro de listas)

Permitem organizar conteúdo em níveis hierárquicos.

  • Frontend
    • HTML
    • CSS
    • JavaScript
  • Backend
    • Node.js
    • Python
    • PHP

📖 Dica extra: Listas de definição (<dl>)

Usadas para glossários ou pares termo/descrição.

HTML
Linguagem de marcação para estruturar páginas web.
CSS
Folhas de estilo para definir aparência visual.
JavaScript
Linguagem de programação para interatividade.
